# note-auto-writer

HIROのnote記事・Instagram投稿・X投稿を半自動で作成するためのリポジトリです。

## 使い方

1. `prompts/hiro_note_prompt.md` にある文体ルールを使う
2. `templates/note_template.md` をもとに記事を作る
3. `drafts/` に下書きを保存する
4. Cursor / Codex / ChatGPT に「このメモからnote記事を作って」と依頼する

## HIROの発信テーマ

- 命
- 希望
- 癒し
- 余白
- 自然
- 潜在意識
- 病気と生き方
- 今日を大切に生きること
