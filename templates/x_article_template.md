# X記事テンプレート

## X投稿の基本ルール

- 140文字以内
- 余白を大切に
- 改行を使って読みやすく
- 強すぎない言葉で、静かに届ける

---

## パターン1：出来事＋気づき

```
[出来事を一文で]

[気づきや感情]

[読者へのやさしいひとこと]
```

### 例
```
廃棄しようとした鉢から
小さな葉っぱが出ていた。

終わったと思っていたものが
また動き始めていた。

諦めなくていい、まだ続いている。
```

---

## パターン2：問いかけ型

```
[問いかけ]

[自分の答えや気づき]

[余韻を残す一文]
```

### 例
```
今日、何かひとつ
「生きていてよかった」と思えましたか？

私は朝の光がカーテンを通る瞬間に
そう思えました。

小さくていい。それで十分。
```

---

## パターン3：詩的な一言

```
[詩的な短文、1〜3行]

[ハッシュタグ（任意）]
```

### 例
```
苦しみの中にも、
必ず光の隙間はある。

#生きること #癒し #今日を生きる
```

---

## よく使うハッシュタグ

```
#生きること
#癒し
#潜在意識
#今日を生きる
#命を諦めない
#自然からの気づき
#HIRO
```
